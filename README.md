* 🎨 Modern UI/UX

* 💼 Professional online presence

## ⚡ Features

### 🎨 UI & Design

* Cyberpunk / neon-inspired interface
* Fully responsive (mobile-first)
* Clean and minimal layout

### 🎬 Animations & Effects

* Particle.js animated background
* Scroll-based animations (AOS)
* Typing animation (dynamic roles)
* 3D tilt effects
* Custom cursor interactions

### 🧠 Smart Sections

* 📊 GitHub stats integration
* 🎓 Experience timeline
* 🏆 Certifications showcase
* 📬 Functional contact form

### 🚀 Extra Features

* ⬆️ Back-to-top button
* ⏳ Preloader animation
* 🚫 Custom 404 page
* 🎮 Hidden Easter egg (Konami code)

## 🚀 Run Locally

```bash
git clone https://github.com/chiragvaze/portfolio.git
cd portfolio
```

```bash
# Run server
python -m http.server 8000
# or
npx live-server public
```

Open:

```
http://localhost:8000
```

## 🎨 Customization

* Update content → `public/index.html`
* Change theme → `public/css/variables.css`
* Modify animations → `public/script.js`
